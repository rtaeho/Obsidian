### OG Tag
- Open Graph Protocol
```html
<head>
	<meta property="og:title" content="멋쟁이사자">
	<meta property="og:description" content="멋쟁이사자처럼과 함께 즐겁게 코딩">
	<meta property="og:image" content="./thumbnail.png">
</head>
```

### 레이아웃
- `<header>` : 머릿말
- `<nav>`
- `<section>` : 내용
- `<footer>`: 저작권, 연락처 등
- 웹 구조가 이해하기 편함 - semantic : 의미가 잘 전될되는 웹
